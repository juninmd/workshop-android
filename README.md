# Workshop-Android

**A standard software project.**

**Installation:**

1.  Clone the repository: `git clone https://github.com/your-username/workshop-android`
2.  Navigate to the project directory: `cd workshop-android`
3.  Install dependencies: `gradle install dependencies`
4.  Run `gradle build` to build the project.

**Usage:**

*   **Building:**  `gradle build`  (Build the project)
*   **Running:** `gradle run` (Execute the application)

*   **Analyzing:** `gradle analyze` (Analyze the code)
